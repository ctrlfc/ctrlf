## Meta 标签

**Doctype**: 声明 HTML5, 需要写在顶部

```html
<!-- Doctype HTML5 -->
<!doctype html>
```

**charset( 字符类型 )**: 正确声明 `charset` meta(UTF-8)

```html
<!-- 设置文档的字符编码>
<meta charset="utf-8">
```

**Viewport( 视口 )**: 正确声明 `viewport` meta

```html
<!-- 响应式网页设计viewpoint声明 -->
<meta name="viewport" content="width=device-width, initial-scale=1">
```
